Please ensure that the CSV file has the following field headings:

  • Object Number (must not have any records with this field empty)
  • Department Name
  • Collection Name (General)
  • Collection Name (Specific)
  • Item Name

For the following headlines for every record there should only be content in one of the similarly named fields, whether it is a Museum item or an Archive item:

  • Extent (Museum Items)
  • Extent (Archive Items)
  • Description (Museum Items)
  • Description (Archive Items)
  • Location of Provenance (Museum Items)
  • Location of Provenance (Archive Items)
  • Date of Provenance (Museum Items)
  • Date of Provenance (Archive Items)

Any other fields with additional information are fine.
